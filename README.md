# آکادمی نووا | وب‌اپ آموزش ارز دیجیتال و هوش مصنوعی

یک وب‌اپلیکیشن راست‌به‌چپ، واکنش‌گرا و مدرن برای معرفی آموزشگاه تخصصی ارز دیجیتال، بلاکچین و هوش مصنوعی. این پروژه به‌صورت استاتیک و بدون نیاز به ابزار build ساخته شده و شامل صفحه فرود کامل، معرفی دوره‌ها، مسیر یادگیری، داشبورد نمونه دانشجو، پلن‌های شهریه و فرم مشاوره است.

## امکانات

- طراحی فارسی و RTL با فونت Vazirmatn
- هدر چسبان، منوی موبایل و ناوبری نرم
- هیروی معرفی آکادمی با آمار، CTA و کارت ترمینال آموزشی
- فیلتر تعاملی دوره‌ها بر اساس حوزه آموزشی
- مسیر یادگیری ۱۲ هفته‌ای برای ترکیب Crypto و AI
- داشبورد نمونه دانشجو برای نمایش پیشرفت و تکالیف
- پلن‌های شهریه و فرم ثبت درخواست مشاوره
- انیمیشن ورود محتوا هنگام اسکرول
- طراحی واکنش‌گرا برای موبایل، تبلت و دسکتاپ

## اجرای محلی

کافی است فایل `index.html` را در مرورگر باز کنید یا یکی از دستورهای زیر را برای اجرای سرور محلی بزنید:

```bash
npm start
```

یا بدون Node.js:

```bash
python3 -m http.server 8000
```

سپس به آدرس زیر بروید:

```text
http://localhost:8000
```

برای توقف سرور، در همان ترمینال کلیدهای `Ctrl + C` را فشار دهید.
